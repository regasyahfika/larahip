## Laravel Vuejs with docker
How to install

- composer install
- npm install
- configuration, cp .env.example .env
- php artisan key:generate
- configuration Database .env
- configuration docker-compose.yml and docker-compose.env
- configuration folder docker/nginx/default.conf
